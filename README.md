# Exclusive E-Commerce 🛒

Figma dizayni asosida React.js da yaratilgan, ko'p tilli (UZ, EN, RU) onlayn do'kon platformasi.

### 🚀 Texnologiyalar
* **Frontend:** React.js (Vite)
* **State:** Redux Toolkit
* **Language:** i18next
* **Bot:** Telegram Integration

### 📁 Loyiha tuzilishi
* `src/pages` — Home, About, Contact, SinglePage.
* `src/redux` — Global state management.
* `root/*.json` — Tarjima fayllari (UZ, EN, RU).

### 🛠 O'rnatish
1. `npm install`
2. `npm run dev`
